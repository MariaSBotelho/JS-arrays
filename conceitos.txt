Arrays ---------------
lista ordenada de valores enumerados em que cada valor é chamado de elemento, 
e cada elemento se localiza numa posição na lista chamada de índice.
valores = elementos 
const numero = [5]; // array com 1 elemento
const nums = [50, 43, 12]; // array com 3 elementos
indice        0  ,1 , 2
length = tamanho de um array -quantidade total de elementos (valores/dados) dentro de um array

Cada elemento está localizado em uma posição na lista, chamada de índice

array esparso - 
existem posições vazias (sem valor definido) entre os índices ocupados.
mesmo add ou removendo um elemento, é respeitado a posição que esta vazia.
